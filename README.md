This dashboard imports processed data from Databricks, where the ETL process was carried out on a sales dataset. During the ETL, the necessary relationships between the tables were established, using the existing keys in each one of them.

-------------------------------------------------------------------------------------------------------------------------------

Este dashboard importa dados processados no Databricks, onde foi realizado o processo de ETL sobre um dataset de vendas. Durante o ETL, foram estabelecidas as relações necessárias entre as tabelas, utilizando as chaves existentes em cada uma delas.
